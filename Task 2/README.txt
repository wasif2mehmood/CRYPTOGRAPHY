For any mode, i.e., for both Encryption and Decryption, the following command is to be typed:

.\SubstitutionCipher.exe filename.txt

where filename.txt is the name of the input file that is either to be encrypted or decrypted.

1. Once the input file is successfully opened, the user is prompted to choose between encryption and decryption.
2. After which the user enters the Keyword as well as the name of the output file along with its extension.
3. The desired operation is performed and the files are saved as requested.
